How to use web scraping API directly through your browser.

How to Use the API

To scrape content from a website, you simply need to provide the target website's URL as a query parameter in your API endpoint. Here is the format:

https://sim-api-tst.onrender.com/scrape?url=<target-website-url>

Replace <target-website-url> with the actual URL you want to scrape.

Example Usageüõå

https://sim-api-tst.onrender.com/scrape?url=https://google.com

Understanding the Responseüßê

Once you use the link, the API will return a JSON object containing the scraped content. For example:

{
    "data": "Here is the scraped content from the target website."
}

Handling Errors

Common issues include missing the url parameter or providing an invalid say bad website URL... or maybe something You made up that doesn't exist
